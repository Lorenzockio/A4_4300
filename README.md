# A4_4300
This is my implementation of the reaction diffusion simulation. The simulation starts with a single random orgin point. This point is always set so it will remain a continuous source of diffusion. Clicking the screen will set a new "random" origin point based on mouse position. (For the life of me, I can not get it to place the origin at the mouse position). There are eight sliders. Four control the values of feed, kill, diffusionA, and diffusionB. The other four sliders control the orientation of both diffusions by shifting the laplacian weights. Two sliders control left and right directions and the other two control up and down.
